About

This is a minimal, framework-free static site you can use in Microsoft Learn modules to practice deploying to Azure App Service. Edit index.html, styles.css, or script.js, then redeploy.

Key Points

  • No build step required.
  • Deploy with a single az webapp up --html command.
  • Uses Free (F1) Linux App Service Plan by default in examples.
  • Includes basic accessibility & responsive layout.

Try It

  1. Open browser console to see a greeting logged by script.js.
  2. Modify content locally, redeploy, then hard refresh (Ctrl+F5).
  3. Add a new page (e.g. about.html) and link it.
